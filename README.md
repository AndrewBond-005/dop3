Реализовать JSF тег counter - счетчик слов и символов в тексте  

Обязательные атрибуты  
- id - идентификатор элемента  
- show - что показывать (chars, words, lines, all)  
- опциональный excludeSpaces - не считать пробелы (true/false)  

Пример тега  

<text:counter id="counter1" show="all" excludeSpaces="false">  
Это пример текста для подсчета.  
В нём несколько строк.  
И разные слова!  
</text:counter>  
  
  
Вывод будет:  
  
Символов: 67  
Слов: 10  
Строк: 3  
  
[сам текст]
